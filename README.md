<h1 align="center">Prompit</h1>

<p align="center">
  呼之则来的小小翻译工具。<br>
  按一下快捷键，它来；翻完一句话，它走。
</p>

---

> 名字是 *Prompt it* 的合成词——意思就是「给它下个提示」。
> 它也确实只会这个。

Prompit 是一个常驻后台的悬浮小窗。**需要它的时候按一下快捷键**，它在光标旁边冒出来，借助 AI 模型帮你翻一句话（或者顺手改一句话），然后把结果直接塞回光标所在的位置，自己消失。

它干的都是些小活儿——而且，它自己不带模型，**得接一个第三方的 AI 服务才能干活**（见下面「借力 AI」那一节）。

## 它是 / 它不是

先把话说清楚，省得你期待错地方：

| 它是 ✅ | 它不是 ❌ |
|---|---|
| 一个按快捷键才出现的悬浮小窗 | 一个全天候在线、自动冒泡的助手 |
| 一次只回你一句话的"笨"应用 | 一个会自己动脑、自己规划、自己执行一连串任务的 AI agent |
| 翻译为主、轻技能为辅的小工具 | 一个要接管你打字的 AI 输入法 |
| 输入法归输入法，翻译归翻译 | 不会替换、不会拦截你的系统输入法 |

简单说：**给它一句话，它回一句话。多一步它都不会自己走。**

### 为什么不是 AI agent？

AI agent 会自己上网查资料、自己决定下一步、自己把一串事干完。Prompit 不会。

它每次只做一件事——把你给它的文字，按你选的方式变个样（翻译、润色……），然后把结果塞回你刚才光标所在的地方。干完，收工，消失。它不思考人生，也不替你思考人生。

### 为什么不是 AI 输入法？

AI 输入法会一直挂在你打字的地方，每个字都想过一遍。很多人不想要这种"全天候陪伴"。

Prompit 走的是另一个路子：

- **平时根本不存在**——你不按快捷键，它就在后台安静待着，不挡你、不看你打字。
- **翻译和输入法井水不犯河水**——你想用系统输入法就用，需要翻译时再单独把 Prompit 叫出来。
- **想要才来，不想要就彻底走**——用完按 `Esc`，它就消失了。
- **自定义比输入法多得多**——快捷键、目标语言、翻译风格、词典、轻技能，全都能自己改（见下面[自定义](#自定义)）。

## 功能亮点

### 🪂 呼之则来，挥之即去
按 `Alt+Y`（可改），小窗在光标旁边弹出；翻完按 `Enter`，结果直接粘回你刚才的位置，小窗自动消失。不用切换窗口，不用复制粘贴。

### 🌐 翻译为主
- 多种目标语言，还能自己加、自己排序。
- **翻译人格**：想让翻译听起来像程序员写的、像正式公文、还是像在聊天？加个人格就行。
- **用户词典**：有些词你想固定怎么翻（比如人名、术语），丢进词典，它每次都听你的。

### ✨ 轻技能为辅
除了翻译，还能干点别的"一句话变身"小活儿，比如**润色**——把你写的同一门语言改得更顺。

这些小活儿叫"轻技能"。你可以自己写、自己改，还能用 Markdown 文件导入导出，换电脑也带着走。

### 🔌 借力 AI，但模型你自己挑
Prompit 的功能是靠 AI 模型实现的，**但它自己不带模型**——你得接一个第三方的 AI 服务（比如 OpenAI、DeepSeek、Kimi、智谱、阿里云……常见的 20 多家都有预设），接好才能用。

为什么不自带？因为模型是要花钱的，谁来付这笔账都不合适。所以模型归你挑、账单归你管，Prompit 只负责把话传过去、把结果接回来。

**关于钱，说清楚两件事：**

- **Prompit 目前完全免费**，没有付费功能、没有内购、没有订阅。软件是开源的（Apache-2.0），你想用就用。
- **你接的那个 AI 服务可能要钱**——这取决于你选的供应方和模型。很多供应方有免费额度，本地跑的模型也行。挑个**便宜、轻量**的模型就够了，Prompit 干的都是小活儿，没必要大材小用。

> 钱包会感谢你的。

## 下载安装

去 [Releases 页面](https://github.com/erelief/Prompit/releases) 下载对应系统的安装包：

- **Windows**：`.msi` 或 `.exe`
- **macOS**：`.dmg`
- **Linux**：`.deb` 或 `.AppImage`

装好打开后，第一次会让你接一个模型供应方——没模型它就是个空壳。接好就能用了。

## 快速上手

1. **装好打开**，跟着首次设置接一个模型供应方（挑个便宜的）。
2. **关掉设置窗口**。
3. **按 `Alt + Y`**——小窗冒出来，输入要翻译的话，按 `Enter`，结果直接粘到光标处。

就这样。三步。

> 💡 第一次不知道快捷键？开机时它会弹个小提示告诉你。忘了也没关系，设置里随时能看、能改。

## 自定义

不想用默认的？几乎都能改：

- **快捷键**：呼出（`Alt+Y`）、切换模式（`Alt+M`）等，全部可重新录制。
- **外观**：亮色 / 暗色 / 跟随系统、透明度、字号。
- **目标语言**：内置常用语言，还能自己加、自己排序。
- **翻译人格**：自定义翻译风格。
- **用户词典**：固定某些词的译法，支持导入导出。
- **轻技能**：自己写、自己改、Markdown 导入导出。
- **开机启动**、**发送后是否自动关闭**、**历史记录条数**……

## 常见疑问

**它会偷看我打字吗？**
不会。你不按快捷键，它就在后台待着，不碰你的键盘、不接管你的输入法。只有你主动叫它，它才动。

**它能帮我自动完成一串任务吗（比如"查资料→总结→发邮件"）？**
不能。它一次只回一句话，不会自己规划、不会自己连着干好几步。想要那种，去找 agent，别找它。

**它是不是要替换我的输入法？**
不是。输入法归输入法，翻译归翻译。你照常用系统输入法打字，需要翻译时再单独叫它出来。

**我的数据存哪儿？**
存你本机，加密保护。不上传到 Prompit 的服务器（因为压根没有这服务器）。还能导出加密备份、换电脑时导回来。

**我一定要付费的模型吗？**
Prompit 本身不要钱，但你接的那个 AI 服务可能要——这取决于你选的供应方和模型。很多供应方有免费额度，本地跑的模型也行。挑个便宜的轻量模型就够用了。

## 面向开发者

<details>
<summary>技术栈 & 本地开发（点开）</summary>

Prompit 用 Tauri 2 + Vue 3 + TypeScript 构建，跨平台（Windows / macOS / Linux）。

```bash
# 装依赖
npm install

# 本地开发运行
npm run tauri dev

# 打包
npm run tauri build
```

需要 Node.js 和 Rust 工具链。
